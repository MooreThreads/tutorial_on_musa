
1. 如何查询ContainerToolkit版本？
```
dpkg -l |grep container

#输入如下：
ii  containerd                                 1.6.12-0ubuntu1~20.04.3               amd64        daemon to control runC
ii  docker.io                                  20.10.21-0ubuntu1~20.04.2             amd64        Linux container runtime
ii  mt-container-toolkit                       1.8.0-1                               amd64        MT Container Toolkit
```
